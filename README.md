## 🖥️ Network Monitor

Dashboard web para monitoramento de sites e serviços em tempo real.

### 📌 Features
- Monitoramento de disponibilidade (online/offline)
- Medição de tempo de resposta (ms)
- Atualização automática de status
- Interface responsiva
